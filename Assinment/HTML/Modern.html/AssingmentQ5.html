Difference between HTML vs HTML5

Sr.No HTML HTML5
1 Uses <font>, <center>, and other styling tags for formatting. Uses CSS for styling and layout, encouraging separation of structure and style.
2 No native support for audio and video embedding. Introduces <audio> and <video> tags for easy multimedia embedding.
3 Does not support semantic tags like <header>, <footer>, or <section>. Includes new semantic tags that improve document structure and accessibility.
