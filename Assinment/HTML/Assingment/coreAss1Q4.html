HTML Tags

  1. <body> - Contains the visible content of the document, such as text, images, and links.
  2. <h1> to <h6> - Define header tags, with <h1> being the most important (largest) and <h6> the least important (smallest).
  3. <p> - Defines a paragraph of text. Browsers automatically add some space (margin) before and after each <p> element.
  4. <a> - Defines a hyperlink, which is used to link to other web pages or sections within the same page. The href attribute specifies the URL.
  5. <img> - Embeds an image in the document. The src attribute specifies the path to the image, and alt provides alternative text.
